# NAME
	orm64
	
# SYNOPSIS
	orm64 ...
	
# DESCRIPTION
	Fantasy operating system or what I call it now: an API layer for creating Software with Lua.
	
	help - Lists commands
	script - Run Lua code

# EXAMPLES
	$ orm64 
	Simply starts the Orm64 shell.
	
	$ orm64 help
	Lists commands avaliable for Orm64
	
	$ orm64 script "print(32)"
	Prints 32 using Lua
